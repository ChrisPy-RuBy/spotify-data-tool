## Summary

Simple tool for looking at the data dump that spotify provides

## Brief


She wants to be able to...
- Upload a zipped folder 
- Introspect all playlists
- Introspect tracks 
- Overview Analytics section
	- Most common track in terms of playlist 
	- Most common track in terms of number of plays
- The results should be displayed in a UI such as a web-browser
- The user is not technical and so CLIs and TUIs are probably not suitable
- The results should look nice
